Lexical Structure

  • Lexical Structure is "the set of elementary rules that specify how you write programs in a language".
  • Examples of Lexical Structure
    • what variable names look like
    • delimiter characters for comments
    • program statement separation
  • Structure requires Unicode character set
  • Case Sensitive
  • Whitespace & Line Breaks are ignored
  • Comments
    • Any text between // and the end of the line
    • Any text between the characters /* and */

    • A literal is a data value that appears directly in a program
    • Examples of literals
      • Numbers
      • A string of text

      • Null
